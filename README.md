# 🔐 Página de Login

Uma interface moderna e responsiva de **login e cadastro**, desenvolvida com **HTML, CSS e JavaScript**, com foco em experiência do usuário, design visual e fundamentos de front-end.

🔗 **Acesse o projeto:**  
https://vittfs.github.io/login-vit/

---

## 📌 Visão Geral

Este projeto é uma **interface de autenticação front-end**, criada para simular um fluxo real de login e cadastro.

O objetivo foi desenvolver uma **UI limpa, intuitiva e visualmente atrativa**, aplicando boas práticas de layout, estilização e interatividade.

--- 

### ⚙️ Implementação

- **HTML5** — estrutura semântica dos formulários  
- **CSS3** — estilização, layout e responsividade  
- **JavaScript** — interações e comportamento da interface  

Principais funcionalidades:
- Campos de entrada (e-mail e senha)  
- Alternância entre login e cadastro  
- Efeitos de hover e foco  
- Transições suaves na interface  

---
